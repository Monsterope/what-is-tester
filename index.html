Tester คือ

คนที่มีหน้าที่ตรวจสอบคุณภาพของ software ที่ถูกผลิตขึ้นมาโดย programmer ด้วยใช้วิธีการทดสอบต่างๆ
เพื่อให้มั่นใจว่า software ที่จะไปถึงมือลูกค้า ตรงตามความต้องการของลูกค้า มีคุณภาพและปราศจากข้อผิดพลาด (Defects)

Testing

ในกระบวนการทดสอบต้องทำอะไรบ้าง

Test plan

First

คือการวางแผนการทดสอบว่ามี ขอบเขตอะไร เงื่อนไข จุดประสงค์ เป้าหมาย ระยะเวลา และวิธีการทำงานต่างๆ

Test case

Second

เอกสารที่บอกขั้นตอนหรือเงื่อนไขต่างๆของซอฟต์แวร์ที่เราทดสอบนั้นๆ ตรงตามความต้องการของลูกค้าหรือไม่

Test Defect
